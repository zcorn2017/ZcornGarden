# 体系

## PCR扩增ZZA片段*2管（例）
> [!WARNING]
> 以下为一个典型例子
> 应当遵循水、引物、酶的顺序

- 按照顺序添加如下试剂：

|试剂|体积($ul$)|
|----|----|
|ddH~2~O|20|
|ZZA-R|2|
|ZZA-F|2|
|Template(ZZA-T Vector)|1|
|Prime|ZZA-T Vector|
|Total|50|

- 轻轻吹吸后离心去除气泡

# PCR程序
## PCR仪使用方法
> 请注意PCR仪内板子的孔径大小，以免放错
1. 打开电源按钮，等待开机。
2. 按照说明编辑循环步骤
3. 设定在12摄氏度无限循环

## 典型例子
- 以下循环33次     

| 温度（摄氏度） | 时间 |
| -------------- | ---- |
| 98             | 10s  |
| 55             | 5s   |
| 72             | 15s  |
- 最后设置在12度无限循环

# 跑胶鉴定（可选）
- 取5$ul$加入2$ul$ 6* loading跑胶鉴定
- 条带正确后使用柱式PCR产物回收试剂盒纯化回收PCR产物
